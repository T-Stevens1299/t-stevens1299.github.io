Prison Break

Overview

Prison Break is a 3D platformer game that utilises possession mechanics.


The aim of the game is to escape a prison by defeating prison guards and possessing drones and cars to solve puzzles.


The main focus of this project for me however was learning how to import animations into unreal engine and utilise them in scripts.



❮ ❯



Game Mechanics

Possessable Drone:

One of the main mechanics of the game is the drone that you can possess. This is controllable from terminals around the map and can move about, activate pressure plates and shoot enemies.

Mechanics Demonstration

This video demonstrates the drone that can be possessed, and the fact that with a button press you can exit the control of the drone anywhere and go back to using the main player.


Animations:

This was the main thing I learned developing this project, so I tried to implement as many animations as I could, such as sword drawing and holstering and various attack animations.

AI Enemies:

In order to increase the complexity of the AI enemies, I utilised Unreal Engines behaviour tree system to add various states such as attacking the player and hunting for them.
